# 仿真结果判读指南 - 如何处理“超标”红点

## 1. 为什么 Max 值总是超标？

在有限元仿真（ANSYS Maxwell）中，经常会出现最大电场计算值（Max E）远超空气击穿阈值（30 kV/cm 或 3e6 V/m）的情况，例如即便在这个优化后的模型中，仍然显示 `1.123e8 V/m`。

**这通常不是设计缺陷，而是“数值奇异点 (Singularity)”。**

### 原理解析
*   **尖角效应**：数学上，只要存在直角（90度）或尖锐边缘（半径=0），该点的电场强度理论上是无穷大。
*   **网格相关性**：网格分得越细，软件算出来的最大值就越高（逼近无穷大）。这个最大值没有实际物理意义，只是数学上的极限。

---

## 2. 如何在报告中解释？(最佳实践)

只要您的“红点”区域极小（仅限于几个网格节点），就可以在报告中判定为**合格**。请使用以下方法来论证：

### 方法一：体积排除法 (Volume Clipping) —— 最具说服力
证明“超标区域”的体积微乎其微，不足以引发击穿。

1.  在 Maxwell 中绘制电场云图。
2.  双击云图标尺 (Legend)。
3.  在 **Scale** 选项卡中，把 **Max** 值手动设为 `3e6` (3x10^6 V/m，即空气击穿值)。
4.  观察云图：
    *   如果只看到几个孤立的“红点”：说明只是数值奇异点，忽略。
    *   如果看到**大片连通的红色区域**：说明真的设计不合格，需要倒角或增加距离。

**报告话术：**
> “局部最大场强出现在金属棱边尖角处，通过 Reciprocal 标尺观察，超过 3 kV/mm 的区域仅限于棱边 0.1mm 范围内，且未形成贯通性击穿通道。根据工程经验，此类微观尖峰场强属于数值计算奇异点，不会导致宏观绝缘失效。”

### 方法二：路径探测法 (Line Probe)
证明离开尖端 1mm 之后的电场是安全的。

1.  画一条线 (Line)，从高场强点出发，延伸到空气中 10mm。
2.  做 XY Plot (Field along Line)。
3.  您会看到曲线在 x=0 处极高，但瞬间掉落。

**报告话术：**
> “沿关键放电路径进行场强扫描，虽然触头边缘表面场强较高，但在离开表面 1mm 处已迅速衰减至 1.5 kV/mm 以下。平均场强远低于空气击穿阈值。”

## 3. 如果非要消除那个 1e8 的数字

如果您必须要给出一个“全绿”的截图来让领导或客户满意，可以使用**“显示截断 (Display Truncation)”**技巧：

1.  双击云图 Legend。
2.  **Scale** 选项卡 -> 取消勾选 Auto。
3.  将 **Max** 手动填为 `3000000` (3e6)。
4.  点 Apply。
5.  此时所有超过 3e6 的地方都会变成同样的深红色。
6.  (可选) 如果不想看红色，可以将 Color Map 的最高颜色改为橙色或黄色（在 `Color Map` 编辑里）。

---

## 4. 结论

**对于您的当前结果 1.12e8 V/m：**
*   **物理上**：它是数值计算导致的虚高值。
*   **工程上**：只要它不连成一片，**您的设计就是合格的**。不用为了这个数字去无限倒圆角（倒圆角会显著增加网格量和计算时间）。

建议直接使用**方法一**调整云图上限为 `3e6`，然后截图作为最终报告的依据。
