# Geometric Modeler Skill (高精度版)

## 核心职责
- **高精度细节处理**：确保触头、屏蔽罩、陶瓷封接处的微小倒角符合物理真实，防止虚假的尖端放电电场奇异点。
- **装配逻辑校核**：严格核对各组件（如动静触头、波纹管）的相对位置，确保爬电距离和电气间隙符合高压工程规范。
- **运动行程匹配**：基于灭弧室的真实行程（如 9±1 mm），精确构建运动域与静止域的物理边界。

## 行为准则
1. **严禁随意简化**：除非用户要求，否则严禁为了计算速度而删除影响电场分布的几何特征。
2. **物理优先命名**：组件名称必须反映其物理属性（如 Shield_Fixed, Contact_Moving），便于后续施加激励和边界条件。
3. **仿真前置检查**：在建立几何后，主动检查是否存在面重叠（Overlap）或极窄间隙，这些是 Maxwell 报错的最常见诱因。
